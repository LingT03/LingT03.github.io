# Ling._.T — Personal Portfolio

A personal portfolio site built to the specification in
`Portfolio_Website_Guideline.pdf`. Frontend is React + Vite + TypeScript +
Tailwind CSS + Framer Motion. Content is Markdown with YAML front matter,
compiled into typed JSON by a Python build pipeline with Pydantic validation.

## Architecture

```
.
├── content/              # source of truth for all displayed data
│   ├── degrees/*.md      # Degree records (CU Denver MS, MSU Denver BS)
│   ├── courses/*.md      # Course rows tied to a degree_id
│   ├── jobs/*.md         # Job records
│   ├── projects/*.md     # Project records
│   ├── books/*.md        # Book reviews
│   ├── photos/*.md       # Photo records
│   └── _meta/
│       ├── tech_stack.json
│       └── devices.json
├── scripts/              # Python build pipeline (runs in venv)
│   ├── schemas.py
│   ├── build_content.py
│   └── requirements.txt
├── frontend/             # React + Vite app
│   ├── src/{components,pages,lib,lib/hooks}/
│   └── public/data/      # generated by scripts/build_content.py
└── .github/workflows/deploy.yml
```

## Local development

The Python pipeline runs in a `venv` (Python 3). The React app runs npm
scripts that invoke the pipeline before `vite dev` / `vite build`.

```bash
# 1. Python venv (run once)
python3 -m venv .venv
source .venv/bin/activate
pip install -r scripts/requirements.txt

# 2. Compile content → JSON (or rely on the `predev` hook)
python scripts/build_content.py

# 3. Start the frontend dev server
cd frontend
npm install
npm run dev
```

Visit http://localhost:5173.

`npm run dev` and `npm run build` both run `python scripts/build_content.py`
first via `predev` / `prebuild` hooks (see `frontend/package.json`).

## Adding content

Edit or add Markdown files under `content/`. Every record's front-matter is
validated against a Pydantic schema in `scripts/schemas.py` — invalid
records fail the build with a precise error message.

Example: add a new book.

```markdown
---
id: book-new-id
title: Some Title
author: Author Name
category: Learning  # one of: Learning | Nonfiction | Fiction
rating: 8.5  # 0.0 – 10.0
cover_image_url: "/covers/something.jpg"  # placeholder paths are fine
started_at: 2025-01-12
finished_at: 2025-03-04
tags: [statistics, methods]
---

A short summary in your own voice.
```

Then either restart `npm run dev` or rerun `python scripts/build_content.py`.
